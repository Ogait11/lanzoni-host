Perguntas Frequentes

O que é uma Key FiveM?

A Key é uma licença oficial do FiveM que permite seu servidor funcionar e acessar benefícios exclusivos como roupas EUP, server list colorida e muito mais.

Preciso de Element Club?

Para keys acima de 48 slots, é necessário ter uma assinatura ativa do Element Club do FiveM. Nós configuramos tudo para você.

Quanto tempo leva para ativar?

A ativação é feita em até 24 horas após a confirmação do pagamento e envio das informações necessárias.
